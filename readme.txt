== Description ==
Über vyble® Recruiting
Recruiting leicht gemacht: Die vyble® HR-Plattform hilft Unternehmen, ihren Recruiting-Prozess zu digitalisieren und effizienter zu machen. Über unsere Plattform können Sie Stellenausschreibungen verwalten, Bewerberdaten erfassen und das Onboarding neuer Mitarbeiter mit wenigen Klicks organisieren, damit Sie bei der Suche nach neuen Mitarbeitern wettbewerbsfähig bleiben.

Die Hauptfunktionen von dem vyble® Recruiting Plugin sind:
- automatische Synchronisation von Stellendaten aus dem vyble® Recruiting System zu WordPress
- Anzeige einer Stellenübersicht mit Verlinkungen zu den Stellen-Detailseiten
- Generierung von Stellenseiten inkl. Stellenbeschreibungen und Formular zum Empfangen von Bewerbungen
- Senden aller Bewerbungen zum vyble® Recruiting System
- individuelle Designeinstellungen für die Stellenübersicht und  Stellen-Detailseiten
